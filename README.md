# TP2-React-App-Multiples-Paginas
Una aplicación de recetas de múltiples páginas en la que se puede buscar por nombre o ingrediente, agregar a favoritos y visualizar los detalles de las recetas. La página también cuenta con traducciones dinámicas a español e inglés.

### Cómo usar la App

* **Sin instalación**
    * Visualizar la página en Vercel → https://tp-2-react-app-multiples-paginas.vercel.app/

* **Con instalación**

1- Instalar **Node.js** → https://nodejs.org/es

2- Clonar el repositorio 
```
git clone https://github.com/LuciaScalia/TP2-React-App-Multiples-Paginas.git
```

3- Correr el siguiente comando en consola
```
npm install
```

4- Correr el siguiente comando en consola para moverse a la carpeta correcta
```
cd tp2-react-pwa
```

5- Correr el siguiente comando en consola para iniciar la aplicación
```
npm start
```
